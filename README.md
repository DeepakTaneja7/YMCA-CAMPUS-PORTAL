# 🎓 YMCA Campus Care Portal

A full-stack web application designed to streamline campus maintenance and issue reporting for J.C. Bose University (YMCA Faridabad). This portal bridges the gap between students and department heads, allowing for seamless complaint filing, tracking, and resolution.

## ✨ Features
* **Role-Based Authentication:** Secure login and registration for both Students and HODs (Heads of Department) using bcrypt password hashing.
* **Interactive Dashboards:** Dedicated views for students to track their complaint history and for HODs to manage department-specific issues.
* **Dynamic Issue Reporting:** Students can file detailed complaints (categorized by department, priority, and location) with real-time UI updates.
* **Secure Database Architecture:** Relational data management ensuring complaints are strictly tied to verified student enrollments.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Single Page Application architecture)
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Security:** bcrypt (password hashing), CORS, dotenv (environment variable management)
